Quick start: Modyfying a map with Blender

In this guide, we'll walk you through the process of modyfying a FRAIL map file with blender

1. Requirements & Preparations

First, make sure that you have appropriate Blender version installed. Due to frequent API changes, our plugin is only guaranteed to work on Blender's Foundation Blender 2.68

Then, you should copy blender plugin script into your installation's addons directory (by default, it's in C:\Program Files (x86)\Blender Foundation\Blender\2.68\scripts\addons). Go ahead and copy the FRAIL_IMPORT directory.

The result should look somewhat like this (depending on different plugins you may have installed before):

You're all set!

If all went well, it's time to open Blender. Skip the opening panel and go straight into User preferences pane. You should be presented with view like this:

Type "frail" into the search box and check the only option into "activated" mode.

2. Importing a map

Now, locate "Info" section in one of the panels (Blender's layout may be confusing when you see it for the first time, yeah) and choose File -> Import -> FRAIL map file (.json)

Next, navigate to mod_levels directory (frail_pwr\build\data\mod_levels) and choose map desired for moding. Import it!

Caution: it will take... some time. Lots of time, actually, especially with arena_a. Just wait patiently. You may open developer console (Window -> Toggle system console) if you want to stay informed of what's going on in the background.

When the map finishes importing you're all set! Well, almost.

Press N and in "Display" section check the "Textured Solid" option

You're all set!

3. Playing around

For the time being, you are only permitted to repositioning and deleting object from map. Sorry! Upgrade is on the way.

Try it out yourself! Move that pesky lava lake to a middle of the map, set the perfect trap for Boss, or align a chain of barrels to trap actors in a sea of fire and crowd control. You don't need snares? Remove them alogether!

4. Saving changes

When you're done, proceed to File -> Export -> FRAIL map file (.json)

Navigate do levels directory (sic!). You'll find it in frail_pwr/build/data/levels. Save under desired name. Note that plugin will automatically add a "_mod" suffix to your choosen name

Click "Export FRAIL .json map file", and you're all set!

5. Nota bene

1. Don't try to import another map directly after exporing current one. Blender has... issues with removing meshes.

2. When in doubt, don't be afraid to take a look in the developer console Window -> Toggle system console to learn what went wrong

3. While adding is not yet supported, .obj files for all game objects are available in frail_pwr/build/data/mod_levels/game_objects. You may try importing one manually to set its location wherever you please, and manually modyfying .json map file later, adding a fitting entry.
